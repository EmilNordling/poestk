## Goals with poestk
- [ ] Mobile friendly, both UI and performance wise
- [ ] Exposed a nice API for poesk's Passive Skill Tree
- [ ] Released as an Progressive Web Application
  
## Build
Development
```sh
$ yarn dev
```

Production
```sh
$ yarn build && yarn start
```
